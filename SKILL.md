## 3. How to use

### 调用方式

```bash
# 基础排课（当前时间）
python3 ~/.openclaw/skills/liuren-skill/scripts/liuren_pan.py

# 指定时间排课
python3 ~/.openclaw/skills/liuren-skill/scripts/liuren_pan.py --date "2026-04-03 15:00"

# 指定问事类型
python3 ~/.openclaw/skills/liuren-skill/scripts/liuren_pan.py --date "2026-04-03 15:00" --question "失物"

# 输出 JSON 格式
python3 ~/.openclaw/skills/liuren-skill/scripts/liuren_pan.py --date "2026-04-03 15:00" --json
```

### 命令行参数

| 参数 | 简写 | 说明 |
|------|------|------|
| --date | -d | 占问时间 (YYYY-MM-DD HH:MM) |
| --question | -q | 问事类型（失物/婚姻/事业/出行/疾病/官司/求财/考试） |
| --json | -j | 输出 JSON 格式 |
