# 大六壬技能

> "大六壬，三式之最。" —— 传统术数

大六壬是中国古代最高层次的占卜术数之一，与奇门遁甲、太乙神数合称"三式"。以月将加时、起四课三传、排天将神煞，进行断课分析。本技能提供完整的大六壬排盘和断课功能。

## ✨ 核心功能

- ✨ 月将加时起课
- ✨ 四课排布（干阳、干阴、支阳、支阴）
- ✨ 三传推算（初传、中传、末传）
- ✨ 天将排布（贵人、螣蛇、朱雀等 12 天将）
- ✨ 神煞查询（天乙贵人、文昌、桃花等）
- ✨ 五行生克分析
- ✨ 自动化断课（吉凶判断、断语）
- ✨ 应期推算
- ✨ 纯 Python/JavaScript 实现，无外部依赖

## 🔧 技术特点

- 🔧 精确月将计算（按节气）
- 🔧 完整四课三传算法
- 🔧 天将排布规则
- 🔧 纯 JavaScript 实现，无外部依赖
- 🔧 支持 JSON 格式输出

## 📖 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dglijin-oss/liuren-skill.git
cd liuren-skill

# 确保 Node.js 14+ 已安装
node --version

# 赋予执行权限
chmod +x index.js
```

### 基本使用

```bash
# 当前时间起课
node index.js

# 指定时间起课
node index.js --date "2026-04-10 15:00"

# 指定问事类型（带断课分析）
node index.js --date "2026-04-10 15:00" --question 财运

# 输出 JSON 格式
node index.js --date "2026-04-10 15:00" --json
```

### 命令行参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| --date | -d | 日期时间 (YYYY-MM-DD HH:MM) | 当前时间 |
| --question | -q | 问事类型 | 通用 |
| --json | -j | 输出 JSON 格式 | 否 |

### 问事类型

| 类型 | 说明 |
|------|------|
| 通用 | 综合判断 |
| 财运 | 求财、投资、生意 |
| 事业 | 工作、升职、创业 |
| 婚姻 | 恋爱、婚姻、感情 |
| 健康 | 疾病、健康、养生 |
| 官司 | 诉讼、纠纷、是非 |
| 出行 | 外出、旅行、迁移 |
| 求子 | 怀孕、生育、子女 |
| 失物 | 寻物、找人 |
| 考试 | 考试、面试、竞争 |
| 交易 | 买卖、合作、签约 |

### 四课推导

1. 日干寄宫 → 干阳
2. 干阳上神 → 干阴
3. 日支 → 支阳
4. 支阳上神 → 支阴

### 三传推导

- 初传：四课中克贼最多者
- 中传：初传所生或所克者
- 末传：中传所生或所克者

### 常见课体

| 课体 | 特征 | 含义 |
|------|------|------|
| 元首课 | 一上克下 | 事从上来，主动 |
| 重审课 | 一下克上 | 事从下来，被动 |
| 知一课 | 只有一克 | 事有专一 |
| 涉害课 | 多克 | 事复杂，需权衡 |
| 昴星课 | 无克 | 事不明，待时 |
| 伏吟课 | 天地同 | 事停滞，不动 |
| 反吟课 | 天地冲 | 事反复，变动 |
